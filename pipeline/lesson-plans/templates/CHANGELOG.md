# Template changelog

Each entry records a version of the session-plan template. Historical plans
can be regenerated against the template that was current at their delivery
date. See `generate.py`'s `TEMPLATE_VERSION` constant.

## v1.0 — 2026-04-14

- Initial superset template for CommunityPlus AMEP delivery.
- Covers VRQA accredited-course alignment, Standards for RTOs 2025 (ASQA),
  AMEP Deed touchpoints, CommunityPlus audit trail, and assessment-chain
  integrity (Mapping + Assessor Guide).
