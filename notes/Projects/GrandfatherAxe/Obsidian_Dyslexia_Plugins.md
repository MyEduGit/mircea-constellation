## Set OpenDyslexic Font
After installing Style Settings:
1. Settings → Style Settings
2. Set Interface Font to **OpenDyslexic**
3. Set Line Height to **1.8–2.0**
4. Set Font Size to **18px** or larger

Download OpenDyslexic free from opendyslexic.org
