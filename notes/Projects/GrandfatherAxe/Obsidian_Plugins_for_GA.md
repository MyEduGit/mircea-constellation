# Obsidian Plugins for Grandfather's Axe (April 2026)

**Goal:** Make GA automatic, visual, rewarding and dead-simple for Mircea (70, dyslexia, memory support).

## Top 6 Plugins (install in this order)

| # | Plugin Name | Why it helps GA | Difficulty |
|---|-------------|-----------------|------------|
| 1 | **Templater** | Auto-fills GA Memory template v3 | Easy |
| 2 | **QuickAdd** | One-keystroke "Capture Idea" or "New GA Log" | Easy |
| 3 | **Dataview** | GA logs into dashboards & streaks | Easy |
| 4 | **Tasks** | Tracks actions from GA drift reports | Easy |
| 5 | **Improved Mermaid** | Better Mermaid editor + toolbar | Very Easy |
| 6 | **Periodic Notes** | Automatic daily/weekly GA summary | Very Easy |

**Also consider:**
- **Obsidian Git** → automatic backup of every GA log
- **Kanban** → visual board for your Idea Inbox
- **Local REST API** → fast n8n → Obsidian writing

## How to install
1. Open Obsidian → Settings (gear icon)
2. Community plugins → Browse
3. Search each name → Install → Enable
4. Restart Obsidian once
