# Grandfather's Axe (GA) — The System We Are Building

**Name origin:** "Grandfather's axe had 5 handles and 3 heads replaced… yet it is still the same axe."  
**Goal:** Stop drifting. Give Mircea (70) an external brain that watches where he spends his time.

## What GA does every hour
- Watches work across apps/platforms
- Generates Mermaid charts
- Stores charts in Obsidian
- Claude bot reads charts → tells you: "On track" or "Drifting sideways"
- Reports in plain language
- Logs behavioral patterns (Trigger / Reaction / Vector / Value / Design note)

## Core features we will build
1. n8n workflow on OpenClaw (already running)
2. Hourly Mermaid timeline
3. Idea inbox (capture fast, prioritise later)
4. Reward system for long focus sessions
5. Breadcrumb navigation (never get lost in rabbit holes)
6. iPhone/macOS app later
