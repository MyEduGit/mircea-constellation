## Application / Description

This unit describes the performance outcomes, skills and knowledge required by EAL learners to access and identify information and services relevant to personal settlement needs.

This unit is designed for integration and contextualisation with language units to support achievement of English language knowledge and skills.

The outcomes described in this unit relate to the Australian Core Skills Framework (ACSF) and contribute to ACSF indicators of competence for Oral Communication and Reading at Level 1, as well as ISLPR (International Second Language Proficiency Ratings) descriptors for Speaking 1+, Listening 1+, and Reading 1+.
