# Install Messenger on iMac M4 — 2026 Guide

**Timestamp:** 2026-04-13 — AEDT  
**Device:** iMac_M4

## Overview

Meta discontinued the standalone Messenger desktop app for macOS as of late 2025. On an iMac with M4, you have two functional alternatives: creating a Safari-based "Web App" or installing the iPad version of Messenger.

---

## Method 1 — Safari Web App (Recommended)

This method provides a native-like experience with a Dock icon and separate window.

1. Open Safari.
2. Navigate to `www.facebook.com/messages`.
3. Log in to your Messenger account.
4. In the Safari menu bar, select **File → Add to Dock…**
5. Rename it "Messenger" if desired and click **Add**.
6. A Messenger icon appears in your Dock; it opens in a standalone window with notifications.

---

## Method 2 — iPad App on Mac

Apple Silicon allows iPad apps to run on your M4 iMac.

1. Open the **App Store**.
2. Click your **Profile** at bottom-left.
3. Select **iPhone & iPad Apps** at the top.
4. Search for **Messenger**.
5. If previously downloaded on iOS with the same Apple ID, click the **Download/Cloud** icon.
6. The iPad version installs; note it's designed for touch, but functions on Mac.
