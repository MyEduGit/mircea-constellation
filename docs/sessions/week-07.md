## Monday, 9 March 2026

**Theme:** Complaints & problem-solving in transactions

### Session 1 (5:15 - 6:30pm)

**Macroskills: Listening, Speaking**

| | Detail |
|---|--------|
| **Objectives** | Develop language for making complaints and resolving problems in transactions |
| **Units** | VU23519 Element 2 (Cert II) / VU23524 Element 2 (Cert III) |
| **Activities** | **Listening:** Listen to complaint dialogues (in a shop, on the phone) - identify the problem, the request, the outcome. **Speaking:** Role-play pairs: Customer & Shop Assistant - practise complaining politely and responding; Cert III: Telephone complaint about a service |
| **Resources** | Complaint role-play cue cards; Polite complaint phrase cards; Telephone complaint scripts |
| **Language Focus** | Cert II: "I'm sorry, but this doesn't work"; "Can I exchange it?"; Cert III: "I'm not satisfied with the service"; "I'd like to speak to a manager"; Modals: "could," "would," "should" |

### Session 2 (6:45 - 8:30pm)

**Macroskills: Reading, Writing**

| | Detail |
|---|--------|
| **Objectives** | Read complaint responses. Write a complaint message |
| **Units** | VU23519 / VU23524 / VU23558 (Cert I) |
| **Activities** | **Reading:** Read a sample response to a complaint (email/letter) - was the problem solved?; Cert I: Read a simple "How to complain" poster. **Writing:** Cert I: Fill in a complaint form (name, problem, what you want); Cert II: Write a short note complaining about a product; Cert III: Write a formal complaint email with clear structure (problem, evidence, requested action) |
| **Resources** | Sample complaint responses; Complaint form templates; Writing frames |
| **Language Focus** | "I purchased [item] on [date]"; "The problem is..."; "I would like a refund/exchange/repair" |

## Wednesday, 11 March 2026

**Theme:** Arranging services & appointments

### Session 1 (5:15 - 6:30pm)

**Macroskills: Listening, Speaking**

| | Detail |
|---|--------|
| **Objectives** | Practise making appointments and arranging services by phone and in person |
| **Units** | VU23519 Element 2 (Cert II) / VU23524 Element 2 (Cert III) / VU23558 (Cert I) |
| **Activities** | **Listening:** Listen to phone calls making appointments (doctor, plumber, real estate); identify date, time, purpose. **Speaking:** Role-play: calling to book an appointment; Cert III: arranging house repairs with a tradesperson |
| **Resources** | Appointment booking scripts; Phone dialogue cards; Calendar/diary for noting appointments |
| **Language Focus** | "I'd like to make an appointment for..."; "Is [day] available?"; "Can you come on...?"; "What time suits you?" |

### Session 2 (6:45 - 8:30pm)

**Macroskills: Writing, Digital Skills**

| | Detail |
|---|--------|
| **Objectives** | Write appointment confirmation messages. Use digital tools for scheduling |
| **Units** | VU23519 / VU23524 + VU23556 / VU23560 / ICTICT214 |
| **Activities** | **Writing:** Write a confirmation message for an appointment (graded by cert level). **Digital Skills:** Cert I - use phone/tablet to look up a phone number for a service; Cert II - add an appointment to an online calendar (Google Calendar demo); Cert III - write a confirmation email for house repairs, including date/time/address; explore online booking systems |
| **Resources** | Confirmation message templates; Google Calendar demo; Computer lab |
| **Language Focus** | "To confirm, my appointment is on [date] at [time]"; "Please let me know if this needs to change"; "I look forward to..." |
