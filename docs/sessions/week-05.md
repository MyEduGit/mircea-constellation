## Wednesday, 25 February 2026

**Theme:** Assessment catch-up & introduction to transactions

### Session 1 (5:15 - 6:30pm)

**Macroskills: Speaking, Listening**

| Objectives | Resources | Skills |
|------------|-----------|--------|
| Complete remaining Task A/1 assessments. Begin introducing transaction language | Assessment materials (catch-up); Transaction dialogue audio clips; Shopping vocabulary picture cards | **Units:** VU23519 Element 2 intro (Cert II) / VU23524 Element 2 intro (Cert III)<br><br>**Activities:** **Speaking:** Complete any remaining assessments from Monday; Students who have finished: pair work introducing transaction vocabulary - "How much is this?"; "I'd like to return this." **Listening:** Listen to model transaction dialogues (at a shop, on the phone, at a service counter)<br><br>**Language Focus:** "How much...?"; "I'd like to..."; "Can I please...?"; "Do you have...?" |

**Macroskills: Reading, Writing**

| Objectives | Resources | Skills |
|------------|-----------|--------|
| Read simple transactional texts (receipts, notices, forms). Write short transactional messages | Sample receipts, bills, return policies; Form templates; Writing frames | **Units:** VU23519 Element 2 (Cert II) / VU23524 Element 2 (Cert III) / VU23558 (Cert I)<br><br>**Activities:** **Reading:** Cert I: Read a simple receipt - identify item, price, date; Cert II: Read a return policy notice; Cert III: Read a utility bill - identify account number, amount due, due date. **Writing:** Cert I: Fill in a simple form (name, address, phone); Cert II: Write a short note explaining a problem with a purchase; Cert III: Write a message requesting a bill extension<br><br>**Language Focus:** "The total is..."; "I bought this on [date]"; "It doesn't work because..."; "I would like to request..." |
