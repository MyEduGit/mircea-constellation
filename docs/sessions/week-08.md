## Monday, 16 March 2026

**Theme:** Transaction fluency - complex structures & conditionals

### Session 1 (5:15 - 6:30pm)

**Macroskills: Listening, Speaking**

| | Detail |
|---|--------|
| **Objectives** | Develop fluency in extended transactions. Use conditionals and complex sentence structures |
| **Units** | VU23519 Element 2 (Cert II) / VU23524 Element 2 KE (Cert III) |
| **Activities** | **Listening:** Listen to extended transaction dialogues - identify how speakers explain problems in detail, negotiate, and confirm outcomes. **Speaking:** Fluency carousel: rotate through 4 transaction scenarios (3 mins each, increase detail each rotation); Cert III: practise using conditionals in negotiation ("If you can't fix it by Friday, I'll need to...") |
| **Resources** | Transaction scenario carousel cards; Conditional sentence frames; Negotiation phrase bank |
| **Language Focus** | Cert II: Explaining problems in detail: "First... then... so..."; Cert III: Conditionals: "If you can extend it, I'll pay on..."; "Unless the repairs are done by..., I will..."; Past perfect: "I had already called about this" |

### Session 2 (6:45 - 8:30pm)

**Macroskills: Reading, Writing, Digital Skills**

| | Detail |
|---|--------|
| **Objectives** | Read and respond to service-related correspondence. Practise formal digital communication |
| **Units** | VU23519 / VU23524 + VU23556 / VU23560 / ICTICT214 |
| **Activities** | **Reading:** Read a letter/email from a utility company or landlord - identify what action is needed. **Writing:** Write a response (graded): Cert I: fill in a reply form; Cert II: write a short reply; Cert III: write a detailed email response with conditional language. **Digital Skills:** Cert I - practise copy-paste text between apps; Cert II - format an email reply (subject line, greeting, body, sign-off); Cert III - create a complaint tracking spreadsheet (date, issue, action, result) |
| **Resources** | Sample service letters/emails; Reply templates; Computer lab |
| **Language Focus** | "Thank you for your letter/email"; "I would like to..."; "Please contact me on..." |
