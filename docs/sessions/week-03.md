## Monday, 9 February 2026

**Theme:** Casual Conversations - Openings, turn-taking & topics

### Session 1 (5:15 - 6:30pm)

**Macroskills: Listening, Speaking**

| | Detail |
|---|--------|
| **Objectives** | Identify features of casual conversation: openings, closings, topic shifts, turn-taking. Practise initiating and responding |
| **Units** | VU23519 Element 1 (Cert II) / VU23524 Element 1 (Cert III) |
| **Activities** | **Listening:** Listen to 3 model conversations (graded by cert level) - identify who starts, topic changes, how it ends. **Speaking:** Mingle activity - move around room, start 3 short conversations with different partners; Cert III: discuss a simple current topic (e.g., weather, weekend plans) |
| **Resources** | Audio/video clips of casual conversations; Conversation starter cards |
| **Language Focus** | Cert II: "Hi, how's it going?"; "What did you do on the weekend?"; Cert III: "Have you heard about...?"; "What do you think of...?" |

### Session 2 (6:45 - 8:30pm)

**Macroskills: Reading, Writing**

| | Detail |
|---|--------|
| **Objectives** | Read short dialogue scripts. Write a simple conversation script with a partner |
| **Units** | VU23519 (Cert II) / VU23524 (Cert III) / VU23558 (Cert I) |
| **Activities** | **Reading:** Read model dialogue scripts; identify greeting phrases, questions, and responses. **Writing:** Cert I: fill in blanks in a simple dialogue; Cert II: write a 6-line conversation between two friends; Cert III: write a 10-line conversation including topic change and closing |
| **Resources** | Dialogue gap-fill worksheets; Model conversation scripts (graded) |
| **Language Focus** | Question forms: "Do you...?"; "Have you...?"; Connectives: "and," "but," "so"; Turn-taking: "What about you?" |

## Wednesday, 11 February 2026

**Theme:** Talking about daily life & community

### Session 1 (5:15 - 6:30pm)

**Macroskills: Listening, Speaking**

| | Detail |
|---|--------|
| **Objectives** | Sustain a casual conversation about familiar topics (daily routines, family, neighbourhood). Practise giving short responses and asking follow-up questions |
| **Units** | VU23519 Element 1 (Cert II) / VU23524 Element 1 (Cert III) |
| **Activities** | **Listening:** Listen to a conversation about someone's daily routine - answer comprehension questions. **Speaking:** Information gap activity: Partner A has a daily schedule, Partner B asks questions to complete theirs; Cert III: discuss a local community issue (e.g., new park, road works) |
| **Resources** | Daily routine picture cards; Information gap worksheets; Local news clipping (simplified) |
| **Language Focus** | Time expressions: "in the morning," "after lunch"; Cert II: simple present tense; Cert III: present perfect "I've lived here for..." |

### Session 2 (6:45 - 8:30pm)

**Macroskills: Reading, Digital Skills**

| | Detail |
|---|--------|
| **Objectives** | Read about local community services and events. Use digital tools to find information |
| **Units** | VU23558 (Cert I) + VU23556 / VU23560 / ICTICT214 |
| **Activities** | **Reading:** Cert I: Read a simplified council newsletter (identify key services); Cert II: Read a community notice board - identify 3 useful services; Cert III: Read a local news article and summarise. **Digital Skills:** Cert I - practise typing name and address; Cert II - search for local community centre, copy address into email; Cert III - find a local event online, create a flyer in Word/Docs |
| **Resources** | Simplified council newsletters; Community noticeboard images; Computer lab |
| **Language Focus** | "This service is for..."; "It is located at..."; "The event is on [date] at [time]" |
