## Monday, 16 February 2026

**Theme:** Phrasal verbs & discourse markers in conversation

### Session 1 (5:15 - 6:30pm)

**Macroskills: Listening, Speaking**

| | Detail |
|---|--------|
| **Objectives** | Recognise and use common phrasal verbs in conversation. Use discourse markers to manage turn-taking |
| **Units** | VU23519 Element 1 KE (Cert II) / VU23524 Element 1 KE (Cert III) |
| **Activities** | **Listening:** Listen to conversations and identify phrasal verbs used (gap-fill exercise). **Speaking:** Card matching game - match phrasal verb to meaning, then use in a sentence; Role-play: pairs practise short exchanges using target phrasal verbs |
| **Resources** | Phrasal verb flashcards; Gap-fill listening worksheets; Role-play scenario cards |
| **Language Focus** | Cert II: "pick up," "look after," "put on," "get on"; Cert III: "put up with," "look into," "get on with," "come across" |

### Session 2 (6:45 - 8:30pm)

**Macroskills: Reading, Writing, Digital Skills**

| | Detail |
|---|--------|
| **Objectives** | Read texts containing phrasal verbs. Write sentences using target vocabulary. Digital: practise typing and formatting |
| **Units** | VU23519 / VU23524 KE + VU23556 / VU23560 / ICTICT214 |
| **Activities** | **Reading:** Read a short story/dialogue - underline all phrasal verbs; match to definitions. **Writing:** Write 5 sentences using phrasal verbs from today's lesson. **Digital Skills:** Cert I - type 5 phrasal verbs and their meanings; Cert II - type sentences into an email draft; Cert III - create a phrasal verb reference table in Word/Docs (3 columns: verb, meaning, example) |
| **Resources** | Graded reading texts with phrasal verbs; Phrasal verb reference handout; Computer lab |
| **Language Focus** | Discourse markers: Cert II: "by the way," "anyway"; Cert III: "actually," "I mean," "as I was saying" |
