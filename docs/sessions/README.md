1. **Before each session:** Open the relevant `week-XX.md` for your daily plan

3. **Standing resources:** Reference `standing-resources.md` for always-available AMEP PDFs and ESL sites
4. **Unit questions:** Refer to the unit mapping docs for elements, PCs, and assessment detail
5. **After session:** Fill in the "Reflections and Notes" table at the bottom of each plan
