## Monday, 23 March 2026

**Theme:** ASSESSMENT - Task B/2 (Transactions)

### Session 1 (5:15 - 6:30pm)

**Macroskills: Speaking (Assessment)**

| | Detail |
|---|--------|
| **Objectives** | Conduct Assessment Task B (Cert II) / Task 2 (Cert III). Record evidence on PC checklists |
| **Units** | VU23519 Task B (Cert II) / VU23524 Task 2 (Cert III) |
| **Activities** | **Speaking (Assessment):** Cert II - Task B: Role-play as a customer requesting a refund for a faulty item from a sales assistant; Cert III - Task 2: Role-play requesting a bill extension for electricity, or arranging house repairs. Assessor records verbatim evidence on PC 2.1-2.6 / 2.1-2.5 checklists |
| **Resources** | Assessment Task B/2 Cue Cards; PC Checklists (Element 2); Audio recorder; KE Checklist |
| **Language Focus** | Cert II: "I bought this [item] and it's [problem]"; "Can I get a refund please?"; Cert III: "Hi, I'd like to request an extension to pay my electricity bill"; "If you can't extend it, I'll need to..." |

**Macroskills: Listening, Reading, Digital Skills**

| | Detail |
|---|--------|
| **Objectives** | Cert I settlement services continuation. Post-assessment digital consolidation |
| **Units** | VU23558 (Cert I) + VU23556 / VU23560 / ICTICT214 |
| **Activities** | **Listening:** Cert I: Listen to descriptions of transport services - buses, trains, trams; answer simple comprehension Qs. **Reading:** Cert I: Read a transport timetable - find times and destinations. **Digital Skills:** Cert I - search for bus times on PTV website; Cert II (post-assessment) - email self-reflection to teacher; Cert III (post-assessment) - update portfolio with assessment reflection, backup file to cloud |
