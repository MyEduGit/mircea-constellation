# KEY QUESTIONS

The questions Mircea should be asking — grounded in the current state of the
constellation, organized so each answer moves the Mission forward.

> *Mission: "The Urantia Book reaching every human being."*
> Filter for every question below: **Truth · Beauty · Goodness**, and
> the Lucifer Test — *Transparent? Honest? Within mandate? Serves mission?*

---

## 1. Mission — what "reaching every human being" actually requires

1. **What is the next human being who does not yet have access?** Name a
   region, a language, a medium. The abstraction "every human" must
   resolve to a specific person the constellation has not yet served.
2. **Which language is the next bottleneck?** Romanian is in flight via
   scribeclaw. After Romanian, which language maximises reach per unit
   effort — Spanish, Mandarin, Arabic, Hindi, Swahili?
3. **What does success look like in 12 months?** A number of readers,
   a number of languages, a number of papers indexed in Cognee, or a
   qualitative shift (e.g. Gabriel answering Paper-level questions on
   urantipedia.org)?
4. **Where is the Mission leaking effort into self-serving work?** Every
   task that is not traceable back to the Mission is cost, not progress.
   What gets cut this week?

## 2. PhD — Triune Monism as evidence, not decoration

1. **What is the single testable claim of Triune Monism right now?** If
   a reader asked "prove matter, mind, and spirit are irreducible",
   what is the strongest paragraph we can hand them today?
2. **Which Urantia Book paper most directly grounds the thesis, and
   which most challenges it?** Both must be named. An argument that
   ignores its strongest objection is not a thesis.
3. **What does Claude vs. ChatGPT disagreement reveal?** Where the two
   AI partners diverge on a passage is often where the real work is.
   Are those disagreements being logged and resolved, or discarded?
4. **When is the defensible draft due, and what is the minimum shape it
   must have by then?** Without a date, there is no forcing function.

## 3. Governance — UrantiOS, Fireclaw, LuciferiClaw

1. **Has any agent in the fleet failed the Lucifer Test in the last 30
   days?** If the answer is "we don't know", the audit log is not yet
   doing its job.
2. **What is the first real incident Fireclaw will see, and is the
   rules.yaml ready for it?** Declarative rules are only as good as the
   scenarios they have been tested against.
3. **What would trigger a LuciferiClaw case, and who signs the
   annihilation gate?** The Father Function signature is Mircea's — is
   the signing procedure written down, or still in Mircea's head?
4. **Does every subagent actually inherit the covenant?** Spawn Mandate
   is declared; is it verified? What check proves a child process
   loaded COVENANT.md before acting?
5. **When ambiguity is reversible, is the Three-Values choice being
   reported, or silently made?** Silence defaults to transparency —
   where are those reports going?

## 4. Infrastructure — what is fragile right now

1. **Ollama is `warn`.** Is qwen2.5:32b actually serving Gabriel and
   Cognee, or is it a single-point-of-failure one restart away from
   taking the local-AI path down?
2. **`openclaw_ingest` has 4 stubbed handlers.** Which one blocks the
   next mircea_corpus ingestion, and who writes it?
3. **LobsterBot is a skeleton.** Is name-reservation still on the
   critical path, or has the need dissolved? If dissolved, delete it.
4. **Disk is 46% on OpenClaw, 38% on URANTiOS.** At current growth,
   when does either hit 80%, and what gets archived first?
5. **Bot fleet is 10/11 active.** Which bot is down, why, and is the
   absence load-bearing?
6. **Is there a working restore path?** Backups that have never been
   restored are hope, not backups. When was the last drill?

## 5. Knowledge layer — Cognee + Obsidian + Urantipedia

1. **Are the 477 Obsidian docs one-way notes, or part of a loop?** If
   Cognee ingests them but nothing reads the result back into Obsidian,
   the loop is broken.
2. **What question can Gabriel answer today that it could not answer a
   month ago?** If none, the knowledge graph is growing in size but not
   in capability.
3. **Does urantipedia.org serve the same answer Gabriel does?** Public
   surface and private brain must not drift.
4. **Which 10 questions should Gabriel be able to answer perfectly
   before any public launch?** Define them, test them, gate release on
   them.

## 6. Human layer — AMEP, students, readers

1. **Of the 21 AMEP students, how many have touched the Urantia Book
   material directly?** The hub exists; the encounter is what matters.
2. **What is the feedback channel from a student back to the
   curriculum?** One-way teaching is not a loop.
3. **Who is the first non-Mircea maintainer of any part of this
   stack?** A constellation with a single point of human failure is
   fragile regardless of its architecture.

## 7. Time and self — the questions Mircea asks Mircea

1. **What did I do today that only I could do?** Everything else was
   delegable or deletable.
2. **Where did I ask permission on obvious good?** (Covenant §6 — that
   is a failure of trust, not safety.)
3. **Where did I act without transparency?** Silence defaults to
   transparency. What went unlogged?
4. **Is the Mission pausing for any reason I have not explicitly
   sanctioned?** If yes, it should be resumed or the pause should be
   named.
5. **What is the next thing, right now?** Idleness defaults to
   continuation. Never stop.

---

## How to use this file

- Review at the start of each working session. Pick **one** question
  per domain that is live this week.
- When a question is answered, strike it and replace it with the next
  sharper question underneath. The list should get harder, not longer.
- When a question has been stale for two weeks, that is itself a
  signal — either the question is wrong, or the work is stuck.
